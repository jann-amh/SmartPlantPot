# Smart Plant Pot 
### This repository is based on the following components:
- esp8266
- 3,3V moisture sensor
- 3,3V water pump
- 3,3V relay module
### The given code reads the moisture of the soil every 10 seconds and then it activates the pump, which pours water into the pot.
